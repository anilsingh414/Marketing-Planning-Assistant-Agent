# Marketing Planning Assistant Agent

A lightweight, high-performance multi-agent system built with the Groq SDK and Streamlit to support Python 3.14+.

## Features
- **LiteAgent Framework**: Custom orchestration layer for speed and reliability.
- **Researcher Agent**: Performs real-time web search for market patterns.
- **Competitor Analyst**: Analyzes specific companies and identifies gaps.
- **Scheduler Agent**: Generates a detailed marketing plan and timeline.
- **Premium UI**: Modern dark-mode Streamlit interface.

## Setup

1. **Clone the project**.
2. **Install dependencies**:
   ```bash
   pip install -r requirements.txt
   ```
3. **Configure Environment**:
   ```env
   GROQ_API_KEY=your_groq_api_key
   MODEL_NAME=llama-3.3-70b-versatile
   ```
4. **Run the Application**:
   ```bash
   streamlit run app.py
   ```
